# todomvc
Realtime TodoMVC experience using [Imba](http://imba.io) and [GunDB](http://gundb.io)

You can try it online [HERE](http://todos.loqali.com) (open multiple browser screens)

You need [node](http://nodejs.org/) to run locally:
```bash
npm install
```

Development mode:
```bash
npm start
```

Build for web:
```bash
npm run build
```
